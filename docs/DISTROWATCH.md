## Description

NullSec Linux is a security-focused Linux distribution based on Debian 13 (Trixie) designed for penetration testing, digital forensics, security research, and ethical hacking. It features 135+ pre-installed security tools across 13 specialized categories including cloud security, AI/ML security, hardware hacking, and automotive security.

## Key Features

- **135+ Security Tools**: Comprehensive toolkit for all security domains
- **9 Specialized Editions**: Standard, Minimal, Cloud, AI/ML, Hardware, Automotive, Forensics, Mobile, Live
- **Multi-Architecture**: AMD64, ARM64, RISC-V support
- **Modern Stack**: Kernel 6.8 LTS, Debian 13 base, Wayland + Hyprland
- **NullKia Integration**: Mobile security framework with 18 manufacturer support
- **Custom Tooling**: NullSec proprietary security tools in 15+ languages
