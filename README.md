## üöÄ Quick Start

### System Requirements

- **CPU:** 64-bit (x86_64)
- **RAM:** 2GB minimum, 4GB recommended
- **Storage:** 20GB minimum
- **Graphics:** Any (Wayland/X11 supported)

### Installation

1. Download ISO
2. Create bootable USB: `dd if=nullsec.iso of=/dev/sdX bs=4M`
3. Boot from USB
4. Follow installer

### First Boot

```bash
# Update system
nullsec-update

# Check tools
nullsec-fetch

# Configure themes
nullsec-theme apply NullSec-Dark
```
