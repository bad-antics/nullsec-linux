# NullSec Linux Mirror Infrastructure

## Official Mirrors

### Primary Mirror (GitHub Releases)
- **URL**: https://github.com/bad-antics/nullsec-linux/releases
- **Limit**: 2GB per file (suitable for minimal ISOs)
- **Status**: Active

### SourceForge Mirror (Recommended for large ISOs)
- **URL**: https://sourceforge.net/projects/nullsec-linux/
- **Limit**: Unlimited
- **Status**: Pending setup

### OSDN Mirror (Asia-Pacific)
- **URL**: https://osdn.net/projects/nullsec-linux/
- **Limit**: Unlimited
- **Status**: Pending setup

## Mirror Sync Schedule
- GitHub: Immediate on release
- SourceForge: Within 24 hours
- OSDN: Within 48 hours
